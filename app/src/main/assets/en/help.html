Using the backup

You can backup the data of Periodical as files in the internal memory or on a SD card or other locations.

When you use the backup function for the first time you will be asked to select a location where to save the backup. In this view you can navigate using the app menu. When you have reached the folder where you want to save the backup, press the button at the bottom to allow access. Within this folder the backup will be stored in a file named "Periodical-backup.json.gz".

When you copy the backup data to another device and want to restore it again, select the folder, where this subfolder is located.

Using Periodical

You can change the month using the buttons or by swiping to the left or to the right.

To indicate the first day of the period, just tap the day in the calendar. To remove the entry, tap again.

Colors indicate the meaning of a day:

Red – Days of the period.

Blue – Fertile days.

Yellow – Infertile days.

The calculated day of ovulation is marked with an O. The current day is marked with a circle. If there are three little dots below a day there are additional details noted for this day.

On top of the calendar there are three numbers:

Ø – the average cycle length in days.

↓ – shortest cycle length in days.

↑ – longest cycle length in days.

The calculation is done for two cycles in advance.

If you tap on an entry in the list, the corresponding month is displayed in the calendar.

To use the data backup, a memory card or internal memory must be present.

Meaning of maximum cycle length in the settings:

If you have a pause in your calendar entries you may need to adjust the maximum cycle length setting. If the number of days between two periods is longer than this value, that time period will be ignored.

Example: if you don't record periods for 4 months (about 122 days) and you want to start again in the fifth month, you can set the maximum cycle length to a value of 100 or less, so the 4 months without a period will not be used for the calculation.

The default for this value is 183 (6 months).
