Usando o backup

Você pode fazer backup dos dados do Periodical como arquivos na memória interna ou no cartão SD ou em outros locais.

Quando você usar a função de backup pela primeira vez você será solicitado a selecionar um local onde salvar o backup. Neste modo de exibição você pode navegar usando o menu de aplicativos. Quando você chegar à pasta onde você deseja salvar o backup, pressione o botão na parte inferior para permitir acesso. Dentro dessa pasta, o backup será armazenado em um arquivo chamado "Periodical-backup.json.gz".

Quando você copiar o arquivo de backup para outro dispositivo e quiser restaurá-lo novamente, selecione a pasta onde este o arquivo está localizado.

Usando Periodical

Mude o mês usando os botões ou deslizando para a esquerda ou para a direita.

Para indicar o primeiro dia da menstruação, apenas clique no dia no calendário. Para remover, clique novamente.

Cores indicam o significado de cada dia:

Vermelho – Dias da menstruação.

Azul – Dias férteis.

Amarelo – Dias inférteis.

O dia calculado da ovulação aparece marcado com um O. O dia atual é marcado com um círculo. Se há três pontos abaixo de um dia, há mais detalhes que foram dados para este dia.

No topo do calendário há três números:

Ø – o comprimento médio do ciclo em dias.

↓ – menor duração do ciclo em dias.

↑ – maior duração do ciclo em dias.

O cálculo é feito para dois ciclos de uma vez.

Se clicar em um item da lista, o mês correspondente é mostrado no calendário.

Para usar o backup de dados, um cartão de memória ou memória interna precisa estar disponível.

Medição do comprimento máximo do ciclo nas configurações:

Se você tiver uma pausa nas suas entradas de calendário, você pode precisar ajustar a configuração de comprimento máximo de ciclo. Se o número de dias entre dois períodos for maior do que este valor, esse período de tempo será ignorado.

Exemplo: se não se gravam períodos por 4 meses (cerca de 122 dias) e se você quiser recomeçar no quinto mês, você pode definir o comprimento máximo do ciclo para um valor de 100 ou menos, então os 4 meses sem um período não serão utilizados para o cálculo.

O padrão para este valor é 183 (6 meses).
