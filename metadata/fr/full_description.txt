Cette application estime les jours de fertilité selon la méthode Knaus-Ogino et permet de suivre les symptômes menstruels.

ATTENTION : Il est déconseillé d'utiliser ce calendrier pour éviter une grossesse ! Il existe des méthodes plus précises, comme la mesure de la température basale du corps.

Les données peuvent être sauvegardées sur une carte mémoire ou une mémoire de stockage interne.
